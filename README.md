# REST API Monitoring Robot Portal

## Project goal

A web application for configuring and executing REST API monitoring robots.

## MVP

- Create REST API robot
- Manual execution
- Scheduled execution
- HTTP status validation
- Response-time threshold
- Execution history
- PASS / FAIL / TIMEOUT dashboard

## Technology stack

- Frontend: React
- Backend: FastAPI
- Database: PostgreSQL
- Scheduler: APScheduler
- Containers: Docker Compose
- Cloud: AWS EC2, RDS, ECR and CloudWatch
- CI/CD: GitHub Actions
